========================================================================
    CONSOLE APPLICATION : spimaster Project Overview
========================================================================

This example project demostrates how to use the Cypress Usb-Serial API.

The scope of this project is to demonstrate how to communicate with EEPROM connected to 
SPi on the USB-Serial DVK using the cyusbserial API.


IMPORTANT:
----------
* Update referenses to "lib" and "include" paths before building the application.
* The CyUSBSerialVCP.h is available in "<Installation Directory>\library"
* The cyusbserialVCP.lib & cyusbserialVCP.dll are available in "<Installation Directory>\library\x86"
